What ArgoCD does on every sync
1
Poll Git for changes to argocd/apps/ and the overlay paths each Application references (default: every 3 minutes)
2
Detect drift by comparing the rendered manifests against what's running in the cluster
3
Render manifests by running kustomize build on the overlay path specified in each Application
4
Apply changes to the target namespace. Create new resources, update changed ones, delete removed ones
