How sync works
1
You push a change to Git (e.g., update image tag in k8s/apps/backend/overlays/dev/ )
2
parent-app detects the change via backend-dev Application (which points at that overlay path)
3
ArgoCD runs kustomize build on the overlay, applies the result to the dev namespace (selfHeal + prune)
